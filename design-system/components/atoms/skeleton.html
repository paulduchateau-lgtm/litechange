Skeleton

Etats de chargement animes pour surfaces, avatars et lignes de texte

Section A

Variantes de base

Trois formes primitives : rectangle (images, cartes), cercle (avatars), et ligne de texte (paragraphes).

--rect (rectangle)
--circle (cercle)
--text (ligne de texte)
Section B

Carte composee

Composition type : avatar cercle + lignes de texte + rectangle. Simule le chargement d'une fiche collaborateur ou d'une initiative.

Section C

Patron table

Skeleton applique a une structure de table : en-tete + lignes de donnees. Utile pour les tableaux d'equipes et de suivi.

Section D

Panneau theme sombre

Rendu sur surface olive-900 forcee, simulant un dashboard en theme sombre.
