Glass Panel

Panneau a effet givre avec backdrop-filter pour surfaces elevees et overlays

Note

Exception aux tokens

Les Glass Panels utilisent des valeurs rgba() codees en dur pour le background et le border. C'est necessaire car backdrop-filter exige un fond semi-transparent pour produire l'effet givre. Cette exception est documentee et ne s'applique qu'a ce composant.

Section D

Reference CSS

/* Base — dark theme (default) */
.liteops-GlassPanel {
  background: rgba(255, 255, 255, 0.06);
  backdrop-filter: blur(16px);
  -webkit-backdrop-filter: blur(16px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: var(--liteops-radius-md);
  padding: var(--liteops-space-6);
}

/* Light theme variant */
[data-theme="light"] .liteops-GlassPanel {
  background: rgba(255, 255, 255, 0.65);
  border: 1px solid rgba(0, 0, 0, 0.06);
}

/* Size modifiers */
.liteops-GlassPanel--sm { padding: var(--liteops-space-4); }
.liteops-GlassPanel--md { padding: var(--liteops-space-6); }
.liteops-GlassPanel--lg { padding: var(--liteops-space-8); }
