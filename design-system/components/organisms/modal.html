Modal

Dialog overlay organism with 3 size variants, close button, header/body/footer sections. Composes Button atoms for action bars.

These modals are rendered statically for showcase purposes. In production, the overlay uses position: fixed; inset: 0; z-index: 50 to cover the viewport, centered with flexbox.

Size Variants

Small (400px), Medium (560px), Large (720px)

Petite modale

Contenu compact pour confirmations rapides ou messages courts. La largeur maximale est de 400 pixels.

Modale standard

Taille par defaut pour la plupart des interactions : formulaires, details, confirmations avec contexte. La largeur maximale est de 560 pixels, suffisante pour la majorite des cas d'usage.

Grande modale

Modale large pour les vues de detail, les formulaires complexes ou les contenus enrichis. Largeur maximale de 720 pixels. Utiliser pour les cas necessitant plus d'espace horizontal : tableaux embarques, vues comparatives, previsualisations.
