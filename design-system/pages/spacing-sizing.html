Section C — Gabarits de largeur

Sizing Guidelines

Deux largeurs maximales structurent la mise en page : 1140px pour le contenu global, 680px pour la prose editoriale.

max-width: 1140px — content

Conteneur principal de la page. Tous les elements de contenu (tableaux, grilles de cartes, formulaires) s'inscrivent dans cette largeur.

max-width: 680px — prose

Conteneur de texte editorial. Optimise pour la lisibilite longue forme : rapports, descriptions, texte d'aide. Environ 65 a 75 caracteres par ligne.

Grille de base 4px

Tous les espacements, tailles et positionnements respectent des multiples de 4px. Ci-dessous, un composant KPI place sur la grille.
