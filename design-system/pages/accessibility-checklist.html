Section B — Indicateurs de focus

Focus Indicators

Tous les elements interactifs doivent afficher un indicateur de focus visible. Le style standard utilise un outline de 2px dans la couleur d'accent, avec un offset de 2px.

/* Focus style standard */ :focus-visible { outline: 2px solid var(--liteops-border-focus); outline-offset: 2px; }

Section C — Checklist de conformite

Accessibility Checklist

Liste des exigences d'accessibilite a verifier pour chaque composant et chaque page. Chaque point est conforme dans le design system.

  • ✓
    En-tetes de tableau accessibles
    Tous les <th> portent l'attribut scope="col" ou scope="row".
  • ✓
    La couleur n'est jamais le seul vecteur d'information
    Chaque indicateur colore est toujours accompagne d'un pictogramme, d'un texte ou d'un motif distinctif.
  • ✓
    Labels explicites sur tous les champs
    Chaque input, select et textarea possede un <label> explicite ou un aria-label.
  • ✓
    Styles focus-visible sur tous les elements interactifs
    Outline 2px solid var(--liteops-border-focus), offset 2px. Navigation clavier entierement fonctionnelle.
  • ✓
    Respect de prefers-reduced-motion
    Toutes les animations et transitions sont desactivees lorsque l'utilisateur prefere le mouvement reduit.
  • ✓
    Contraste minimum AA (4.5:1 texte, 3:1 UI)
    Verifie pour toutes les paires texte/fond principales. Exceptions documentees pour les labels tertiaires et statuts sur fond colore.
  • ✓
    Tableaux responsifs
    Tous les tableaux sont enveloppes dans un conteneur overflow-x: auto pour le defilement horizontal.
