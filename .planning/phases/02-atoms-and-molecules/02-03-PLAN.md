- Base styles: `background: rgba(255, 255, 255, 0.06); backdrop-filter: blur(16px); -webkit-backdrop-filter: blur(16px); border: 1px solid rgba(255, 255, 255, 0.1); border-radius: var(--liteops-radius-md); padding: var(--liteops-space-6)`
- Note: glass panels work best on dark backgrounds. The rgba values are intentional here since backdrop-filter needs semi-transparent backgrounds. Document this in the showcase as an exception to the "no hardcoded colors" rule.
- Light theme variant: `background: rgba(255, 255, 255, 0.65); border: 1px solid rgba(0, 0, 0, 0.06)` — adapt via `[data-theme="light"] .liteops-GlassPanel` selector in the token layer
- Sizes: sm `padding: var(--liteops-space-4)`, md `padding: var(--liteops-space-6)`, lg `padding: var(--liteops-space-8)`
- Showcase: show panels on a gradient background (use `--liteops-gradient-hero-green-olive`) to demonstrate the frosted glass effect, show light theme variant on paper background, show nested content examples

**skeleton.html** — Skeleton loading states showcase:
- CSS class naming: `.liteops-Skeleton` (base), `--rect` (rectangular), `--circle` (circular), `--text` (text line)
- Base styles: `background: var(--liteops-surface-tertiary); border-radius: var(--liteops-radius-sm); animation: liteops-skeleton-pulse 1.8s ease-in-out infinite`
- `@keyframes liteops-skeleton-pulse`: from `opacity: 1` to `opacity: 0.4` and back. Use `var(--liteops-easing-in-out)` timing.
- Rectangular: `width: 100%; height: 120px; border-radius: var(--liteops-radius-md)`
- Circular: `width: 48px; height: 48px; border-radius: 50%`
- Text line: `width: 100%; height: 14px; border-radius: 4px`, with text-line groups (3-4 lines, last one shorter at 60% width)
- Showcase: show all 3 variants, a composed skeleton card (avatar circle + 3 text lines), a skeleton table row pattern, dark theme panel
- Respect `prefers-reduced-motion`: `@media (prefers-reduced-motion: reduce) { .liteops-Skeleton { animation: none; } }`

No hardcoded colors except glass panel rgba (documented exception).
