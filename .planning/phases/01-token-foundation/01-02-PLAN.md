**Step 2: Create gradient.json.**

Create `design-system/tokens/src/gradient.json` with named gradient definitions. Per CONTEXT.md: gradients are ONLY for landing hero/marketing sections, never dashboards.

```json
{
  "gradient": {
    "hero-green-olive": {
      "$type": "gradient",
      "$value": "linear-gradient(135deg, #6B8A1A 0%, #1C1D1A 100%)",
      "$description": "Landing hero: green-to-olive. ONLY for marketing surfaces."
    },
    "hero-green-black": {
      "$type": "gradient",
      "$value": "linear-gradient(135deg, #527A10 0%, #101110 100%)",
      "$description": "Landing hero variant: green-to-black. ONLY for marketing surfaces."
    },
    "hero-subtle": {
      "$type": "gradient",
      "$value": "linear-gradient(180deg, rgba(107, 138, 26, 0.08) 0%, rgba(28, 29, 26, 0) 100%)",
      "$description": "Subtle green wash for section backgrounds on light theme."
    }
  }
}
```

Note: DTCG gradient type may not be natively supported by Style Dictionary v5 as a built-in `$type`. If so, use `$type: "other"` or a custom type, and register a custom transform that outputs the gradient value as a CSS custom property string. The key is that the generated CSS produces:
```css
--liteops-gradient-hero-green-olive: linear-gradient(135deg, #6B8A1A 0%, #1C1D1A 100%);
```

**Step 3: Create motion.json.**

Create `design-system/tokens/src/motion.json` with duration, easing, and stagger tokens:

```json
{
  "duration": {
    "instant": { "$type": "duration", "$value": "0ms" },
    "fast": { "$type": "duration", "$value": "120ms" },
    "normal": { "$type": "duration", "$value": "200ms" },
    "slow": { "$type": "duration", "$value": "350ms" },
    "slower": { "$type": "duration", "$value": "500ms" },
    "slowest": { "$type": "duration", "$value": "800ms" }
  },
  "easing": {
    "default": { "$type": "cubicBezier", "$value": [0.4, 0, 0.2, 1] },
    "in": { "$type": "cubicBezier", "$value": [0.4, 0, 1, 1] },
    "out": { "$type": "cubicBezier", "$value": [0, 0, 0.2, 1] },
    "in-out": { "$type": "cubicBezier", "$value": [0.4, 0, 0.2, 1] },
    "spring": { "$type": "cubicBezier", "$value": [0.175, 0.885, 0.32, 1.275] }
  },
  "stagger": {
    "item": { "$type": "duration", "$value": "100ms", "$description": "Delay between staggered items. Cap total sequence at 800ms max." },
    "section": { "$type": "duration", "$value": "150ms", "$description": "Delay between staggered page sections." }
  },
  "loop": {
    "pulse-glow": { "$type": "duration", "$value": "4s", "$description": "Observability dot pulse cycle." },
    "blob-drift": { "$type": "duration", "$value": "10s", "$description": "Landing page blob drift cycle. Paused under prefers-reduced-motion." }
  }
}
```

Note: `cubicBezier` is a DTCG type. Style Dictionary v5 should support it. The output should be CSS cubic-bezier() functions:
```css
--liteops-easing-default: cubic-bezier(0.4, 0, 0.2, 1);
```

For duration tokens, the output should be the raw value:
```css
--liteops-duration-fast: 120ms;
```
